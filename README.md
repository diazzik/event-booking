# EventHub

Веб-приложение для бронирования билетов на мероприятия с системой личных кабинетов пользователей.

## О проекте

Данный проект разработан в рамках курсовой работы по дисциплине "Разработка веб-приложений".

Приложение позволяет пользователям просматривать список мероприятий, получать подробную информацию о них, регистрироваться в системе, авторизовываться и оформлять заказы на приобретение билетов.

Для администраторов предусмотрены средства управления мероприятиями и пользователями через административную панель Django.

## Основные возможности

### Пользователь

* регистрация и авторизация;
* просмотр списка мероприятий;
* просмотр подробной информации о мероприятии;
* бронирование и покупка билетов;
* просмотр истории заказов;
* работа с личным кабинетом.

### Администратор

* создание мероприятий;
* редактирование мероприятий;
* удаление мероприятий;
* просмотр зарегистрированных пользователей;
* просмотр оформленных заказов;
* просмотр статистики.

## Запуск проекта локально

### Клонирование репозитория

```bash
git clone https://github.com/diazzik/event-booking.git
cd event-booking
```

### Создание виртуального окружения

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

Linux:

```bash
source venv/bin/activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Выполнение миграций

```bash
python manage.py migrate
```

### Запуск сервера разработки

```bash
python manage.py runserver
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```
